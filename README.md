# Gilded Rose

This is my solution to the Gilded Rose refactoring kata exercise.

## Getting started

Install dependencies

```
npm install
```

Run linter

```
npm run lint
```

Run unit tests

```
npm test
```

Run unit tests in watch mode

```
npm run test:watch
```

Run both linter and unit tests

```
npm run test:all
```

Run test coverage

```
npm run test:coverage
```

Open coverage report

```
npm run test:report
```
